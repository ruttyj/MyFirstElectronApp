# My First Electron App

An app for storing links built with Vue JS, Vue Router, VueX ORM, Knex JS, Bookshelf JS & Vuetify. 

## Getting Started

You need to have `git`, `node` & `npm` installed in your machine.

## Installation

```bash
$ npm install
$ npm run post install

# One terminal
$ npm run watch
# Other terminal
$ npm run electron
```
